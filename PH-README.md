## 描述
厌倦了为简单的联系表单搭建完整后端？FormFast 让你只需一行代码：

```html
<form action="https://formfast.io/f/YOUR_FORM_ID" method="POST">
  <input type="email" name="email" placeholder="your@email.com">
  <button type="submit">Send</button>
</form>
```
