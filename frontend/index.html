FormFast
功能 定价 文档

一行代码搞定表单提交

无需后端开发，只需一个 API 端点即可接收表单数据。 支持 Webhook 通知、数据导出、垃圾邮件过滤。

index.html 
<form action="https://formfast.io/f/YOUR_FORM_ID" method="POST">
  <input type="email" name="email" placeholder="your@email.com" required>
  <textarea name="message" placeholder="Your message..."></textarea>
  <button type="submit">Send</button>
</form>

快速开始

1. 创建表单

点击"开始使用"按钮，输入表单名称，即可获得专属的表单端点。

2. 在 HTML 中使用

将生成的端点地址填入 form 标签的 action 属性：

<form action="https://formfast.onrender.com/f/YOUR_FORM_ID" method="POST">
  <input type="email" name="email" required>
  <textarea name="message"></textarea>
  <button type="submit">提交</button>
</form>

3. 查看数据

在"我的表单"页面，点击"查看数据"即可看到所有提交记录。

API 文档

创建表单

POST /api/forms 
{
  "name": "联系表单"
}

提交表单

POST /f/{formId}

任何表单数据都会自动保存

查看提交

GET /api/forms/{formId}/submissions

Header: X-API-Key: your-api-key
